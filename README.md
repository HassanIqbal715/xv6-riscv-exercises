# xv6-riscv-exercises
This is the lab project for my OS course at university.

## Objectives
- Understanding how the xv6-riscv OS works.
- Playing with it and adding custom system calls and user programs.
- Doing so by solving the xv6-riscv labs for this repository: https://github.com/mit-pdos/xv6-riscv
Labs are found here under the labs section: https://pdos.csail.mit.edu/6.1810/2025/index.html

## Exercises Breakdown
### Week 1
1. Create your own hello world user program. (Complete)
2. Create your own system call that adds two numbers. (Complete)

### Week 2 Utils Exercises
1. Learn to boot xv6 using qemu. (Complete)
2. Create Sleep user program. (Complete)
3. Create a program that reads from a text file, and prints all the numbers that are divisble by 6 or 7 and are present between these separators ".,/\t\n\r-". (Complete)
4. Create a memory dump. (Complete)
5. Create the linux find function. (Complete)
6. Create exec flag for the find function. (Complete)

### Week 3 File System Exercises
1. Large Files. File size that can consist of 65803 blocks.
2. Create Symbolic Links.
